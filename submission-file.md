## Introduction to Application

Dear Environment is a Flutter-based mobile application that transforms everyday environmental observations into interactive digital postcards. Rather than simply capturing images, the application integrates contextual environmental data such as location, weather conditions, air quality, and time to enrich user-generated content.

Users can capture a photo or select an existing image from their device, after which the application retrieves real-time environmental data through external APIs. This information is then used to generate descriptive text and visual styles, forming the basis of a personalised postcard. The application does not automate the final output; instead, it provides a “Studio” interface where users can creatively edit layouts, adjust images, add decorations, and refine generated captions.

The purpose of this application is to encourage users to actively engage with their surroundings and reflect on environmental conditions in a more meaningful way. By combining data, design, and interaction, Dear Environment repositions environmental information from passive consumption to creative expression. The result is a collection of personalised environmental artefacts that function as both memory records and reflective narratives of everyday life.
